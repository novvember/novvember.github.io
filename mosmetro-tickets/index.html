Стоимость билетов на транспорт в Москве

Данные на 2020 год (тарифы с 01.02.2020).

Визуальный анализ цен на общественном транспорте в Москве. Дает общее представление о стоимости транспорта при разных вариантах его использования: и при редких поездках, и при частых. Можно увидеть, какие билеты выгодные в определенной ситуации, какие нет.

Можно выбрать подходящий билет, зная свое количество поездок в месяц на метро (по горизонтали) и на наземном транспорте (по вертикали). Например, тем, кто ездит только на метро, будет интересна только верхняя строка (ноль поездок на наземном транспорте).

Сервис в процессе создания, еще не готов, как видно.

При загрузке страницы будет генерироваться интерфейс и сразу рассчитываться график по всем билетам.

  • Генерация интерфейса графика.
  • Генерация списка билетного меню с галочками и прочим.
  • Упрощенная генерация самого графика для изначального отображения и тестов.
  • Добавлен расчет одного поля (Единый 60 поездок). При наведении мышкой на нужный участок показывается стоимость для этого случая для этого билета.
  • Добавить все остальные виды билетов в расчет (чтобы получился полноценный уже график).
  • Отображение линий от курсора к осям графика
  • Отображение всплывающей подсказки при клике или наведении курсора с подробной информацией
  • Взаимодействие с галочками и возможность пересоздание графика без некоторых билетов
  • Добавление полей с учетом пересадок
  • В конце хочетя еще сделать отдельную форму с ручным указанием полных данных о всевозможных поездках/пересадках и расчетом различных билетов для этой ситуации.
  • Понятный текст с описанием возможностей и, может, каким-то анализом ткущей ситуации.
  • Создание такого для других годов либо просто сравнение цен с прошлыми годами.
